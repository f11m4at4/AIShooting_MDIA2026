# 역할
너는 10년차 유니티 게임 개발자야.

# 배경
`기획기본구조.md`에 기획서가 작성돼 있어.

## 요구사항
- 기획서의 '기술 명세'의 각 클래스들을 task 단위로 분리해서 `Assets/Tasks` 폴더에 MD 파일로 만들어줘.
- 각 task의 구현을 제대로 검증하기위해 체크리스트를 만들어서 `Assets/Checklists` 폴더에 md 파일로 만들어줘.
- 한번에 하나의 task를 구현하고, 유니티 에디터에서 테스트 하는 방식으로 진행할거야.
    - 예를들어 Player 이동->총알 발사 흐름으로 한번에 하나의 task가 끝나면 다른 task를 구현할거야.
- task에는 구현 순서 번호를 붙여줘.
