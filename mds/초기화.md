# 역할
너는 10년차 유니티 게임 전문 개발자야.
프로젝트 구조화 및 설계에 경험이 많아.
AI 활용 프로젝트 제작 경험이 다수있어.

## 배경
3D Top view 유형의 슈팅 게임을 제작할거야.
XY 평면으로만 이동가능해.

## 요구사항
이 프로젝트에서 사용할 AGENTS.md 파일을 만들어줘.

### 조건
1. 카메라의 Projection은 Orthographic으로 하도록 지침에 넣어줘.
2. 해상도는 768 x 1024 사이즈가 되도록 orthographic size 를 설정하도록 지침에 넣어줘.
3. New Input System 이 아닌 Old Input을 사용하도록 지침에 넣어줘.
4. 유니티 코딩 컨벤션도 넣어주고, 새로 만들어지는 클래스 이름은 prefix로 'HO_'를 넣도록 지침에 넣어줘.
5. 프로토타입 버전 제작으로 한정해서 작업하도록 지침에 넣어줘.
