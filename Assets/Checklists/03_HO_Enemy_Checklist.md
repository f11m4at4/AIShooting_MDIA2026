# 03. HO_Enemy 검증 체크리스트

## 컴파일 확인

- [x] `HO_Enemy` 클래스명이 파일명과 일치한다.
- [x] 새 클래스명에 `HO_` prefix가 붙어 있다.
- [x] Inspector 노출 값은 `[SerializeField] private` 필드를 우선 사용한다.
- [x] Unity Console에 컴파일 오류가 없다.

## 이동 검증

- [x] 적이 생성 후 월드 좌표 기준 `Vector3.down` 방향으로 이동한다.
- [x] 적 이동은 XY 평면에서만 처리되고 XZ 평면 이동이 없다.
- [x] 이동 속도를 Inspector에서 변경하면 Play Mode에서 반영된다.
- [x] 이동 중 Z 위치가 게임플레이 이동 값으로 변하지 않는다.

## 제거 검증

- [x] 화면 아래쪽 제거 Y 값을 Inspector에서 조정할 수 있다.
- [x] 적이 제거 Y 값 아래로 내려가면 자동 제거된다.

## 충돌 및 피격 검증

- [x] 플레이어와 충돌하면 `HO_PlayerController`의 피격 메서드를 호출한다.
- [x] 접촉 대미지를 Inspector에서 조정할 수 있다.
- [x] 플레이어와 충돌한 적은 제거된다.
- [x] 총알에서 적 피격 메서드를 호출하면 적이 제거된다.
- [x] 추적, 회피, 원거리 공격 같은 제외 기능이 포함되지 않았다.
